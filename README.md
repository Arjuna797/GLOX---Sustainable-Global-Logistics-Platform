- Interactive markers with detailed facility information.

- **Eco-Efficiency Metrics**: Real-time graphs showing sustainability performance improvements.
- **Dynamic Charts**: Interactive bar, pie, and area charts using Recharts.

### üì¶ **Smart Shipment Tracking**

## üíª Getting Started

### Prerequisites
- Node.js (v16+)
- npm or yarn

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Arjuna797/GLOX---Sustainable-Global-Logistics-Platform.git
   cd GLOX---Sustainable-Global-Logistics-Platform
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```
