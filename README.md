## üíª Getting Started

### Prerequisites
- Node.js (v16+)
- npm or yarn

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Arjuna797/GLOX---Sustainable-Global-Logistics-Platform.git
   cd GLOX---Sustainable-Global-Logistics-Platform
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```
